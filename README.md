# 运行 Fuse 
首先把整个文件夹git到本地  

**需要将chromedriver与fuse.py放在一个目录下**    

_Fuse 现只支持     
Google Chrome	99.0.4844.51 (Official Build) (64-bit)_



需要安装相关支持库  
`pip install selenium`   
`pip install bs4`
